# Research: Cognitive Load Theory

## Summary of Findings

Cognitive load theory (CLT) is a framework for instructional design developed by **John Sweller**, an educational psychologist at the University of New South Wales, Sydney, Australia. The theory was first formally introduced in his 1988 paper "Cognitive Load During Problem Solving: Effects on Learning," published in *Cognitive Science*.

CLT is grounded in human cognitive architecture -- specifically the relationship between long-term memory (which is vast and essentially unlimited) and working memory (which is severely limited in both capacity and duration). Working memory can typically hold only about 4 to 7 items of novel information at a time, and this information is lost within roughly 20 seconds without rehearsal.

The theory identifies three types of cognitive load that compete for limited working memory resources:

1. **Intrinsic cognitive load**: The inherent complexity of the material itself, determined by "element interactivity" -- the number of elements that must be processed simultaneously to understand the content.
2. **Extraneous cognitive load**: Unnecessary demands imposed by poor instructional design, which do not contribute to learning (e.g., confusing layouts, redundant information, split attention between sources).
3. **Germane cognitive load**: The productive mental effort devoted to building and automating schemas -- the cognitive structures that organise knowledge in long-term memory.

Intrinsic and extraneous load are additive: together they must not exceed working memory capacity, or learning breaks down. Good instructional design aims to minimise extraneous load, manage intrinsic load (e.g., by sequencing simple-to-complex), and thereby free up working memory for germane processing.

## Claim Verification

The workbook states:

> "Cognitive load theory... explains why: working memory is limited, and every additional piece of information competes for attention."

**Verdict: Accurate.** This is a faithful, plain-language summary of the core premise of CLT. The theory's central claim is that working memory is severely limited in capacity and duration, and that when instructional materials introduce too many elements at once, those elements compete for the same constrained working memory resources. The phrase "competes for attention" is a reasonable simplification of what the literature describes as additive demands on working memory capacity. No correction is needed.
