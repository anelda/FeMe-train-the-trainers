# Outcomes Map

## Outcomes → Exercises

### Outcome 1: Design assessments that reveal where learners are struggling, rather than simply ranking them.
- **Bloom's level:** Create (verb: *design*)
- **Exercise:** Activity: What would you change? — Bloom's level: Analyse
- **Exercise:** Activity: Separate signal from noise — Bloom's level: Analyse

### Outcome 2: Distinguish between signal and noise in assessment results, and adjust your design to reduce noise.
- **Bloom's level:** Analyse (verb: *distinguish*) / Apply (verb: *adjust*)
- **Exercise:** Activity: Separate signal from noise — Bloom's level: Analyse

### Outcome 3: Plan frequent, low-stakes assessment points across a training programme, based on evidence that spacing supports learning more effectively than a single high-stakes exam.
- **Bloom's level:** Create (verb: *plan*)
- **Exercise:** Activity: Map your assessment points — Bloom's level: Create

### Outcome 4: Evaluate the difference between recall and recognition, and justify your choice of assessment format.
- **Bloom's level:** Evaluate (verb: *evaluate*, *justify*)
- **Exercise:** Activity: Recognition vs. recall — Bloom's level: Evaluate

### Outcome 5: Draft a transparent assessment introduction that reduces test anxiety, drawing on evidence about how anxiety affects learner performance.
- **Bloom's level:** Create (verb: *draft*)
- **Exercise:** Activity: Write your assessment introduction — Bloom's level: Create

## Exercises → Outcomes

### Exercise: Activity: What would you change?
- **Bloom's level:** Analyse (learners examine their current assessment method and identify what it reveals and misses)
- **Serves outcome:** "Design assessments that reveal where learners are struggling..." — Outcome Bloom's level: Create

### Exercise: Activity: Separate signal from noise
- **Bloom's level:** Analyse (learners analyse an assessment against three diagnostic questions and identify noise)
- **Serves outcome:** "Design assessments that reveal where learners are struggling..." — Outcome Bloom's level: Create
- **Serves outcome:** "Distinguish between signal and noise..." — Outcome Bloom's level: Analyse

### Exercise: Activity: Recognition vs. recall
- **Bloom's level:** Evaluate (learners write both formats, compare what each proves, and justify a choice)
- **Serves outcome:** "Evaluate the difference between recall and recognition..." — Outcome Bloom's level: Evaluate

### Exercise: Activity: Map your assessment points
- **Bloom's level:** Create (learners design additional assessment checkpoints across their programme)
- **Serves outcome:** "Plan frequent, low-stakes assessment points..." — Outcome Bloom's level: Create

### Exercise: Activity: Write your assessment introduction
- **Bloom's level:** Create (learners draft and revise an assessment introduction)
- **Serves outcome:** "Draft a transparent assessment introduction..." — Outcome Bloom's level: Create

### Exercise: Activity: Three non-test assessments
- **Bloom's level:** Create (learners invent three alternative assessment approaches and evaluate self-assessment potential)
- **⚠️ ORPHAN** — this exercise does not clearly serve any listed outcome.

## Bloom's Alignment Issues

- **⚠️ EXERCISE UNDERSHOOTS OUTCOME** — Both "What would you change?" (Analyse) and "Separate signal from noise" (Analyse) undershoot Outcome 1 (Create). Learners critique and analyse existing assessments but never produce a new or redesigned one.
- **Minor:** Outcome 2's "adjust your design to reduce noise" component is only lightly addressed — the exercise asks learners to "jot down one change." Resolves itself if a redesign step is added for Outcome 1.

## Flags for Review

- **Flag 1 (main): ORPHAN exercise — "Three non-test assessments."** No outcome covers alternative assessment formats or self-assessment potential. Suggestion: add a new outcome, e.g. "Identify assessment approaches beyond written tests and quizzes, and evaluate whether they could be used for learner self-assessment."
- **Flag 2: Exercises undershoot Outcome 1.** "What would you change?" and "Separate signal from noise" both operate at Analyse, but Outcome 1 targets Create ("design"). Learners never produce a redesigned assessment. Suggestion: add a redesign step after "Separate signal from noise" (e.g. "Now rewrite or redesign this assessment so that it better separates signal from noise"), or lower Outcome 1 to Analyse.
- **Flag 3 (minor): Outcome 2's "adjust" component lightly addressed.** Resolves itself if Flag 2 is fixed by adding a redesign step.
- **No vague verbs.** All outcome verbs are clear and Bloom's-aligned.
- **No other unmet outcomes.** Outcomes 3, 4, and 5 are well-matched to their exercises.
