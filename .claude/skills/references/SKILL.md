---

Use **parenthetical inline citations** in the text — `(Author, Year)` or `(Author et al., Year)` — and collect full references in a `### References` section at the end of the file, separated by a `---` rule.

## Inline citation examples

```
...one of the most reliable findings in learning science (Cepeda et al., 2006).
```

```
A major review of ten learning techniques (Dunlosky et al., 2013) rated both re-reading and highlighting as "low utility" strategies.
```

## Bottom references section format

```
---

### References

- **"Title of the Paper or Book"**
  Author Name(s).
  [*Journal Name*, vol(issue), pages, year](https://doi.org/...).

- **"Another Paper Title"**
  [Author Name](https://author-profile-url).
  [*Journal Name*, vol(issue), pages, year](https://doi.org/...).
```

- Link the author name to a profile page when one is available.
- The source URL should be a DOI link when possible, otherwise a direct URL (PubMed, publisher page, etc.).
- If only one or two authors, list them by name. For three or more, use "First Author et al." in the inline citation but list all authors in the references section.

## Rules

- Every challengeable fact must have a reference. A "challengeable fact" is any claim a skeptical reader might ask "source?" about — specific numbers, dates, effect sizes, named studies, attributed ideas, or causal statements.
- **Do not invent research findings or statistics.** If you reference a study or finding, it must be something you are confident is real. If you are uncertain, describe the general consensus without a specific citation.
- Every source URL should be valid — not just the name of a publication.
- If you cannot find a credible source for a claim, flag it to the user rather than leaving it unreferenced.

## IMPORTANT 

Once you are finished, use a subagent to execute the voice-and-tone skill on the content. It should pay special attention to the references and make sure that the tone has not gotten overly academic.
