Read `6-activities.md` in the current directory. Create a separate file called `workshop-activities.md` in the same directory with group activities for live classroom workshops. This file is not part of the main pipeline — it is a standalone output used in a different workflow. Do not modify or copy `6-activities.md`.

## What to create

Design 3–6 facilitated group activities that a workshop leader can run in a live classroom setting. These are distinct from the self-study activities in the main chapter — they require interaction between participants.

## Format for workshop-activities.md

```markdown
# Workshop Activities: [Chapter Title]

These activities are designed for a facilitator running a live workshop based on this chapter. Each activity includes timing, group size, materials needed, and step-by-step instructions.

## Activity 1: [Title]

**Time:** [e.g. 15 minutes]
**Group size:** [e.g. pairs, small groups of 3–4, whole group]
**Materials:** [e.g. sticky notes, printed handout, projector]

### Instructions

[Step-by-step instructions for the facilitator. Be specific about what to say, what to ask participants to do, and how to debrief.]

### Facilitator notes

[Tips for running the activity — common pitfalls, what to watch for, how to adapt for different group sizes.]
```

## Guidelines

- Use the **voice-and-tone** skill for tone
- Activities should directly connect to key concepts from the chapter
- Include a mix of activity types: discussions, hands-on exercises, role-plays, gallery walks, think-pair-share, case studies
- Each activity should be runnable in 10–30 minutes
- Include clear debrief/reflection prompts for the facilitator
- Consider different experience levels among participants
- Make materials requirements minimal where possible
