Read `3-glossary.md` in the current directory and add learning outcomes and reflective questions to the top. Write the result to `4-preamble.md`.

## What to add

Add a markdown table at the very top of the file (before any existing content), with two rows:

| Questions | _(A set of reflective questions the reader should hold in mind while reading. These should feel like genuine provocations — things a thoughtful trainer might wrestle with — not comprehension questions.)_ |
| :---- | :---- |
| **Learning outcomes** | _(Action-oriented outcomes using verbs like: explain, identify, recognise, describe, design, adjust, distinguish, interpret. Each outcome should describe something observable the reader will be able to do after working through the section.)_ |

## Questions (5–8)

Read through the content carefully and come up with 5–8 questions. These appear BEFORE the content and should spark curiosity. The answers should be found in the text that follows.

Example:
```
- How do I help students remember what they learn over time?
- How do I help students understand how ideas connect, not just memorise pieces?
- How can I check whether students truly understand — not just think they do?
- What kind of feedback actually helps students improve?
- How do I avoid overwhelming students with too much information at once?
- What common teaching habits might look effective but don't actually support learning?
```

## Learning outcomes (4–6)

Example:
```
- Explain in simple terms how memory and understanding develop through practice and connection-making.
- Identify teaching practices that may create a false sense of understanding.
- Recognise signs of cognitive overload in their own teaching materials.
- Describe what makes feedback useful for student improvement.
- Redesign at least one element of their own lesson or course using one evidence-based learning principle.
```
