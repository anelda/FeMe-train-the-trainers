Read `4-preamble.md` in the current directory and add "remember" callout boxes to highlight key takeaways. Write the result to `5-remember.md`.

## What to do

Read over each section and identify the most important takeaways a reader should retain. Add a REMEMBER box near each key insight — they do not need to appear at the end of a section. Place them wherever they are most relevant and useful.

## Format

```
| REMEMBER | _(A 2–4 sentence summary of the single most important takeaway from the section. Written in plain, direct language. This should be something the reader could revisit quickly as a reminder of the core idea.)_ |
| :---- | :---- |
```

## Guidelines

- Aim for 1–3 REMEMBER boxes per major section, depending on density
- Do not use a header for the REMEMBER box — it is formatted as a table
- Each box should capture ONE key idea, not a summary of everything
- Use the **voice-and-tone** skill — plain, direct language
- Do not modify the rest of the content — only add the REMEMBER boxes
- Do not add a REMEMBER box if the section doesn't have a clear, distinct takeaway worth highlighting
